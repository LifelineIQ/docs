Use **Checks & Equipment** to monitor recurring safety checks and record new checks from templates.

## Steps

<Steps>
  <Step title="Open Checks & Equipment">
    Go to **Safety > Checks & Equipment**.
  </Step>
  <Step title="Review daily checks">
    Check the daily safety panel for items due, completed, or needing follow-up.
  </Step>
  <Step title="Review station and unit checks">
    Check station and unit panels for recurring checklist status.
  </Step>
  <Step title="Open a check">
    Select the check template you need to record.
  </Step>
  <Step title="Complete required items">
    Mark each required item with the appropriate result.
  </Step>
  <Step title="Add notes">
    Add notes or follow-up details when something needs attention.
  </Step>
  <Step title="Save the check">
    Select **Save check**.
  </Step>
</Steps>

## Common problems

### Save check is disabled

Complete all required items before saving.

### A check template is missing

An administrator can manage templates in **Admin > Safety > Check Templates**.

### A check found a vehicle problem

Create a maintenance issue in Fleet so the repair work is tracked.
