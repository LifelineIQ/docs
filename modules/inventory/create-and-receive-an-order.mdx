Use **Orders** to create and track purchase orders for supplies and equipment.

## Create an order

<Steps>
  <Step title="Open Orders">
    Go to **Inventory > Orders**.
  </Step>
  <Step title="Start a new order">
    Select **New Order**.
  </Step>
  <Step title="Add order details">
    Add the vendor, order details, and items that need to be ordered.
  </Step>
  <Step title="Save the order">
    Save the order as a purchase order.
  </Step>
  <Step title="Submit or mark ordered">
    Use the row actions to **Submit for Approval** or **Mark as Ordered**, depending on your agency workflow.
  </Step>
</Steps>

## Receive an order

<Steps>
  <Step title="Find the order">
    Search or filter the Orders list.
  </Step>
  <Step title="Open row actions">
    Use the action menu for the order.
  </Step>
  <Step title="Receive items">
    Select **Receive Items** when the shipment arrives.
  </Step>
  <Step title="Confirm received quantities">
    Confirm what arrived and save the receipt.
  </Step>
</Steps>

## Common problems

### I do not see Receive Items

The order may not be in a status that can be received yet. Review the order status first.

### Accounting needs the order

Use **Export to AP** from the order row actions when accounting export is part of your workflow.
