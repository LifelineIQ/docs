Use **Audits** to verify stock counts and resolve inventory discrepancies.

## Steps

<Steps>
  <Step title="Open Audits">
    Go to **Inventory > Audits**.
  </Step>
  <Step title="Start a count">
    Select **New Audit**.
  </Step>
  <Step title="Choose the count setup">
    Select the count type, count area, count scope, and whether it is a blind count.
  </Step>
  <Step title="Start the count">
    Select **Start Count**.
  </Step>
  <Step title="Enter counted quantities">
    Select the audit, enter counted quantities for each line, add notes when needed, and select **Save**.
  </Step>
  <Step title="Resolve discrepancies">
    For mismatches, choose the outcome, add the reason or approver note, and select **Save Resolution**.
  </Step>
  <Step title="Complete the count">
    Select **Complete Count** after required counts and discrepancy work are finished.
  </Step>
</Steps>

## Export a count

Select the audit, then use **Export** to create a count file.

## Common problems

### Complete Count is disabled

Finish required line counts and discrepancy resolution first.

### The audit has no lines

Make sure inventory stock exists for the selected agency and count area, then start a new count.
