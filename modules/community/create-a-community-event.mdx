Use **Events** to add requested, scheduled, confirmed, or completed community events.

## Steps

<Steps>
  <Step title="Open Events">
    Go to **Community > Events**.
  </Step>
  <Step title="Start the event">
    Select **New event**.
  </Step>
  <Step title="Enter the event details">
    Add the event name, date and time, location, status, and other required event information.
  </Step>
  <Step title="Save the event">
    Save the event. LifelineIQ opens the event record after it is created.
  </Step>
  <Step title="Complete event sections">
    Use the event record sections for details, contracts and payments, surveys, hours, and ICS or IAP information when they apply.
  </Step>
  <Step title="Return to the calendar">
    Use **Back to Events** to return to the event calendar or board.
  </Step>
</Steps>

## Common problems

### I cannot find an event

Search the Events page and check the status filter. Events can also appear on the calendar or board view.

### I am looking for hours, surveys, agreements, or ICS

Open the event record. Those tools live inside the event now, not as separate global Events pages.
