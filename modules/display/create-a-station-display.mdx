Use **Create Display** to build a new station screen.

## Steps

<Steps>
  <Step title="Open Create Display">
    Go to **Display > Create Display**.
  </Step>
  <Step title="Enter display information">
    Add the display name, station, division, theme, and header position.
  </Step>
  <Step title="Choose a starting layout">
    Choose **Blank Display**, a system template, or a custom template.
  </Step>
  <Step title="Create the display">
    Select **Create Display**.
  </Step>
  <Step title="Edit the layout">
    Add widgets, configure the persistent header, and set up screen rotation if needed.
  </Step>
  <Step title="Save">
    Select **Save** after layout changes.
  </Step>
</Steps>

## Common problems

### Save is disabled

Make a layout or settings change first. Save is enabled when the display has unsaved changes.

### I need more than one rotating screen

Enable screen queue behavior and add another screen from the edit page.
