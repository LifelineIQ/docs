Use **New Report** to create a NERIS-compatible fire incident report.

## Steps

<Steps>
  <Step title="Open New Report">
    Go to **Fire Reporting > New Report**.
  </Step>
  <Step title="Choose how to start">
    Select **Blank Report** to start from scratch, or choose **Link to Dispatch Incident** to populate the report from an existing dispatch incident.
  </Step>
  <Step title="Select a dispatch incident if needed">
    Search by address, type, or ID. Select the incident, then choose **Create Report from Selected Incident**.
  </Step>
  <Step title="Complete the report wizard">
    Fill out the required report sections in the Fire Report Editor.
  </Step>
  <Step title="Save and continue">
    Save draft work as you go. Return to **All Reports** or **Drafts** when you need to find it later.
  </Step>
</Steps>

## Common problems

### I cannot find the dispatch incident

Search by address, type, or dispatch ID. If the incident is not available, start with **Blank Report**.

### The report is not ready for review

Open the report editor and finish required NERIS and agency-required fields before moving it forward.
