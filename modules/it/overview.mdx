IT is where users report technical issues and IT staff manage tickets, devices, assignments, attachments, and asset history.

## Common work

    Create a help desk ticket from the IT Help Desk page.
  </Card>
  <Card title="Manage an IT asset" icon="laptop" href="/modules/it/manage-an-it-asset">
    Add, assign, update, and review device or asset records.
  </Card>
</CardGroup>

## Where to go

- Open **IT > Dashboard** for IT overview.
- Open **IT Help Desk** to report an issue or work tickets.
- Open **All Assets** to manage IT assets.
- Open **Admin > Logistics > IT** for IT settings.

<Note>
  Older **Report an Issue** and **My Issues** links redirect to the IT Help Desk.
</Note>
