Use **All Assets** to track devices, equipment, assignments, status, files, and related help desk tickets.

## Add an asset

<Steps>
  <Step title="Open IT Assets">
    Go to **IT > All Assets**.
  </Step>
  <Step title="Start the asset">
    Select **Add Asset**.
  </Step>
  <Step title="Enter details">
    Add the asset name, type, tag or serial information, assignment, location, cost, and status.
  </Step>
  <Step title="Save the asset">
    Save the asset so it appears in the IT Assets list.
  </Step>
</Steps>

## Update an asset

<Steps>
  <Step title="Find the asset">
    Search by name, serial, or tag. Use status and type filters if needed.
  </Step>
  <Step title="Open the asset">
    Select the asset row.
  </Step>
  <Step title="Edit or assign">
    Use **Edit** to change details, or **Assign** to update assignment.
  </Step>
  <Step title="Review related work">
    Check tickets, activity, notes, assignment history, and files.
  </Step>
</Steps>
