Use **IT Help Desk** to report an issue, track your own tickets, or work the agency queue.

## Create a ticket

<Steps>
  <Step title="Open IT Help Desk">
    Go to **IT > IT Help Desk**.
  </Step>
  <Step title="Start the ticket">
    Select **Report an Issue**.
  </Step>
  <Step title="Describe the issue">
    Enter the issue details, category, priority, affected asset, and any useful notes.
  </Step>
  <Step title="Submit the ticket">
    Save the ticket so it appears in the help desk queue.
  </Step>
</Steps>

## Work a ticket

<Steps>
  <Step title="Find the ticket">
    Use the queue scope, status filter, category filter, or ticket search.
  </Step>
  <Step title="Open the ticket">
    Select the ticket number or row.
  </Step>
  <Step title="Update the ticket">
    Change status, priority, assignment, or linked asset as needed.
  </Step>
  <Step title="Add evidence or notes">
    Add a photo, attachment, or comment.
  </Step>
  <Step title="Resolve or close">
    Update the status when the issue is resolved or closed.
  </Step>
</Steps>
