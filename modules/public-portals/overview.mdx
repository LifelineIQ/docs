Public portals are tokenized or agency-specific links used by people outside the main LifelineIQ app.

## Public link types

- Records request forms and request status pages.
- Secure records download portals.
- Facility portals and trip tracking links.
- Payment links.
- Call shift response links.
- Public staffing views.
- Careers pages.
- Community event request, contract, payment, and survey links.

<Warning>
  Do not publish real portal tokens in documentation. Use only generic examples when training users.
</Warning>
