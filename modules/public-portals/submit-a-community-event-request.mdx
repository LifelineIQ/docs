Use the public community event form when someone outside LifelineIQ needs to request an event.

## Steps

<Steps>
  <Step title="Open the event request link">
    Open the agency's community event request form.
  </Step>
  <Step title="Enter event details">
    Complete contact, event, location, date, time, and service details requested by the form.
  </Step>
  <Step title="Review the request">
    Confirm the information is correct.
  </Step>
  <Step title="Submit">
    Submit the request.
  </Step>
  <Step title="Save the request number">
    Keep the request number from the success page for follow-up.
  </Step>
</Steps>

## What happens next

The request appears in LifelineIQ Community event workflows for staff review.
