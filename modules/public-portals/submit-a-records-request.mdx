Use the agency records request link when a requestor needs medical records.

## Submit the request

<Steps>
  <Step title="Open the records request link">
    Open the records request form provided by the agency.
  </Step>
  <Step title="Enter requestor information">
    Complete **Requestor Information** and each required form step.
  </Step>
  <Step title="Upload documents">
    Add required documents in the **Documents** step.
  </Step>
  <Step title="Complete payment if required">
    If payment is required, complete the **Payment** step.
  </Step>
  <Step title="Submit">
    Select **Submit Request**.
  </Step>
</Steps>

## Check status or download records

After submission, use the status link to view request status. If the agency sends a secure records link, verify the requestor email, then use **Download** for available files.

## Common problems

### The link will not open

The agency link may be expired, unavailable, or typed incorrectly. Request a new link from the agency.

### The request shows awaiting payment

Complete the required payment before staff can finish the request.
