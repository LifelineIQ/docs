Use the Safety or Security chart list to create a new ops chart.

## Steps

<Steps>
  <Step title="Open the correct chart type">
    Go to **Ops Charts > Safety Charts** or **Ops Charts > Security Charts**.
  </Step>
  <Step title="Create the chart">
    Select **New Chart**.
  </Step>
  <Step title="Complete the chart details">
    Enter details, location, notes, custom fields, tasks, links, attachments, and closeout information that apply.
  </Step>
  <Step title="Save">
    Select **Save**.
  </Step>
  <Step title="Move status forward">
    Use the available status actions to move the chart through the workflow.
  </Step>
</Steps>

## Common problems

### The chart is not editable

Some chart statuses are read-only. Use the status workflow or QA process if changes are needed.

### I created the wrong type

Create safety charts from the Safety list and security charts from the Security list. The chart type is inferred from where you start.
