Use **Reports** for live module reports and **Report Inbox** for scheduled report files after they run.

## Open a module report

<Steps>
  <Step title="Open Intelligence reports">
    Go to **Intelligence > Reports**.
  </Step>
  <Step title="Choose the module">
    Select the module report area, such as Revenue, Fleet, Inventory, Safety, Education, or Compliance.
  </Step>
  <Step title="Choose the report">
    Select the report you need from the module report view.
  </Step>
  <Step title="Review filters">
    Set the filters or date range that match the question you are answering.
  </Step>
</Steps>

## Download a scheduled report

<Steps>
  <Step title="Open Report Inbox">
    Go to **Intelligence > Report Inbox**.
  </Step>
  <Step title="Find the report file">
    Review the report title, report type, sensitivity label, and delivery details.
  </Step>
  <Step title="Download">
    Select **Download**.
  </Step>
</Steps>

## Common problems

### Download is locked or unavailable

Your role may not allow access to that report or its sensitivity level. Ask an administrator to review your access.

### I do not see the report I expected

Scheduled reports appear in the inbox only after they run and deliver. For live reporting, use **Intelligence > Reports**.
