Use **Goals & Metrics** to define compliance targets and monitor whether the agency is meeting them.

## Steps

<Steps>
  <Step title="Open Goals">
    Go to **Compliance > Goals**.
  </Step>
  <Step title="Add a goal">
    Select **Add Goal**.
  </Step>
  <Step title="Choose the module">
    Select the module that owns the metric.
  </Step>
  <Step title="Choose the metric">
    Select the metric you want to track.
  </Step>
  <Step title="Set the target">
    Choose whether the value should be **At least**, **At most**, or **Exactly** the target value.
  </Step>
  <Step title="Create the goal">
    Select **Create**.
  </Step>
</Steps>

## Review goals

The page groups goals by module and shows whether active goals are met, on track, or need attention.

## Common problems

### I cannot select a metric

Choose a module first. Some modules may not have available metrics.

### A goal looks wrong

Edit the goal target and comparison. If the live value looks wrong, review the source module that feeds the metric.
