Use the **Document Repository** to track agency licenses, waivers, policies, renewals, and other compliance records.

## Steps

<Steps>
  <Step title="Open Documents">
    Go to **Compliance > Documents**.
  </Step>
  <Step title="Search or filter">
    Search by document name, or filter by category.
  </Step>
  <Step title="Add a document">
    Select **Add Document**.
  </Step>
  <Step title="Enter document details">
    Add the document name, category, expiration or renewal information, and notes.
  </Step>
  <Step title="Upload the file">
    Attach the document file when available.
  </Step>
  <Step title="Save">
    Save the document.
  </Step>
</Steps>

## Update a CLIA waiver

The page includes a CLIA waiver panel when that document is needed. Select **Upload CLIA Waiver** or **Update CLIA Waiver**, then save the current file and renewal details.

## Common problems

### A document shows expired or expiring soon

Open the document, upload the renewed file, and update the expiration or renewal date.

### I should not delete the document

Use archive when the record should no longer be active but should remain part of the compliance history.
