Use **Work Orders** to keep maintenance requests moving from report to resolution.

## Steps

<Steps>
  <Step title="Open work orders">
    Go to **Fleet > Work Orders**.
  </Step>
  <Step title="Choose the work type">
    Use the page controls to view vehicle work orders or building work orders.
  </Step>
  <Step title="Find the work order">
    Search by work order details, or filter by status and priority.
  </Step>
  <Step title="Open the work order">
    Select the work order row or card.
  </Step>
  <Step title="Review the details">
    Check the description, linked vehicle or building, priority, status, attachments, parts, labor, and activity.
  </Step>
  <Step title="Update the work">
    Change status as work progresses. Add notes, labor, parts, or attachments as needed.
  </Step>
  <Step title="Close the work order">
    Close the work order only after the issue is resolved and the record is complete.
  </Step>
</Steps>

## What happens next

Fleet dashboards and reports use the work order status to show open work, overdue work, and maintenance load.

## Common problems

### I cannot find a work order

Clear status and priority filters, then search again. Also check whether the work order is under vehicle work orders or building work orders.

### I need a new work order

Use **New Work Order** for planned or known maintenance. Use **Report Issue** for a field-reported problem.
