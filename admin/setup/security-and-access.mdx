Use **Admin > Agency Setup > Security & Access** for agency security settings and audit review.

## What this controls

- Login and authentication settings.
- Session and password policy behavior.
- MFA and security posture where enabled.
- HIPAA audit trail and history review.

## Steps

<Steps>
  <Step title="Open Security & Access">
    Go to **Admin > Agency Setup > Security & Access**.
  </Step>
  <Step title="Review authentication settings">
    Confirm login, password, session, and MFA expectations match agency policy.
  </Step>
  <Step title="Review audit settings">
    Check the audit trail or security history when you need to understand who changed or accessed protected workflows.
  </Step>
  <Step title="Save changes">
    Save only after confirming the effect on staff login and access.
  </Step>
  <Step title="Test access">
    Ask an authorized test user to confirm login and role behavior if the change affects access.
  </Step>
</Steps>

<Warning>
  Do not weaken security settings to work around a single login issue. Fix the user, role, or identity problem directly.
</Warning>
